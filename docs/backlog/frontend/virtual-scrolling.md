# Virtual Scrolling for Large Trees

For projects with hundreds of nodes, use virtual scrolling (svelte-tiny-virtual-list or flat rendering mode). Keenmate/svelte-treeview loads 17,000+ nodes in <100ms using flat rendering.

## Priority

M5 or when tree size becomes a performance issue.
